# Data Sources

## Raw station observations

Stored under `data/raw/stations/`

### Current primary file

- `station_hourly_merged_countrywide_v1.csv`

This is the pooled hourly station dataset used as the main raw station input for the post-processing system.

## Station metadata and manuals

Stored under `data/raw/metadata/`

Current contents include station-related manuals and supporting metadata PDFs used to interpret sensor and installation behavior.

## Baseline forecast source

Planned source: Open-Meteo / ECMWF pull generated by project code.

The `data/external/openmeteo/` folder is intentionally empty for now because forecast pulling is part of the code pipeline that will be built later.

## Reference/helper files

The `data/external/reference/` folder is intentionally empty for now because no separate mapping or helper files have been created yet.

## Reproducibility rule

If a bulky file is not committed directly to the repository, its source or reproduction path must be documented here.
